Troubleshooting

Common Issues & Solutions

"Required status check failed"

Problem: PR cannot be merged due to failed status checks
Solution:

  1. Check the specific job that failed in Actions tab
  2. Review error logs and fix the underlying issue
  3. Push new commit to trigger re-run

"SonarCloud quality gate failed"

Problem: Code quality doesn't meet standards
Solution:

  1. Review SonarCloud report (link in PR)
  2. Fix reported issues (bugs, vulnerabilities, code smells)
  3. Ensure test coverage meets thresholds

"iOS build failed with certificate error"

Problem: iOS signing certificates are missing/expired
Solution:

  1. Verify APPLE_TEAM_ID and certificate secrets are set
  2. Check certificate expiration in Apple Developer Portal
  3. Update expired certificates and re-upload to secrets

"App Center upload failed"

Problem: APK upload to App Center unsuccessful
Solution:

  1. Verify APP_CENTER_TOKEN is valid and not expired
  2. Check App Center app exists and token has access
  3. Build will continue with GitHub artifacts even if App Center fails

"Pipeline timeout"

Problem: Jobs exceed timeout limits
Solution:

  1. Check if dependencies are cached properly
  2. Verify no infinite loops or hanging processes
  3. Consider splitting large jobs or optimizing build steps
